# Chicken ```DELETE```

Used to delete a chicken.

| Method   | Url           |
| -------- | ---------------|
| `DELETE`   | `/chicken/:id` |

### Parameters

#### Path parameter

```id``` <small>number</small>&nbsp;&nbsp;&nbsp;<span style="color: orange">**Required**</span>

> Chicken's id to delete

### HTTP response status codes

| Status   | Description           |
|----------|-----------------------|
|```200``` | OK                    |
|```401``` | Problem in request    |
