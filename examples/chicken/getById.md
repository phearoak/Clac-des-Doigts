# Chicken ```GET```

Used to get a single chicken.

| Method   | Url       |
| -------- | -----------|
| `GET`   | `/chicken` |

### Parameters

#### Path parameter

```id``` <small>number</small>&nbsp;&nbsp;&nbsp;<span style="color: orange">**Required**</span>

> Chicken's id to get

### HTTP response status codes

| Status   | Description           |
|----------|-----------------------|
|```200``` | OK                    |
|```401``` | Problem in request    |
