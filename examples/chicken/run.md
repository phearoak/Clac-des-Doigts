# Chicken ```POST```

Used to increment chicken steps.

| Method   | Url           |
| -------- | ---------------|
| `POST`   | `/chicken/run/:id` |

### Parameters

#### No path parameter required

#### Body parameters

```id``` <small>number</small>&nbsp;&nbsp;&nbsp;<span style="color: orange">**Required**</span>

> Chicken's id

### HTTP response status codes

| Status   | Description           |
|----------|-----------------------|
|```200``` | OK                    |
|```401``` | Problem in request    |
